PRÁCTICA 4: RAMIFICACIÓN Y PODA, PROGRAMACIÓN LINEAL
Asignatura: Algoritmia Básica - Curso 2025/26

REQUISITOS:
Python 3.8+
pip install pulp (pip3 en lab000)

EJECUCIÓN:

1. Todas las pruebas (RECOMENDADO - desde carpeta practica4):
   Windows:  python ejecutar_todas_pruebas.py
   Linux:    chmod +x ejecutar_todas_pruebas.sh && ./ejecutar_todas_pruebas.sh

2. Algoritmos individuales (desde carpeta practica4):
   
   Backtracking (Ramificación y Poda):
   $ python3 formarEquipos.py pruebas/pruebas_memoria.txt resultado_bt.txt
   
   Programación Lineal:
   $ python3 programacion_lineal.py pruebas/pruebas_memoria.txt resultado_pl.txt
   
   Con archivo personalizado:
   $ python3 formarEquipos.py entrada_prueba.txt salida.txt
   $ python3 programacion_lineal.py entrada_prueba.txt salida.txt

3. Pruebas individuales (desde carpeta practica4):
   Solo pruebas principales:      python3 pruebas/ejecutar_pruebas_memoria.py
   Solo casos extremos:           python3 pruebas/pruebas_adicionales.py
   Solo comparativa BT vs PL:     python3 pruebas/comparar_algoritmos.py

ESTRUCTURA:
* formarEquipos.py          - Solución con Ramificación y Poda
* programacion_lineal.py    - Solución con Programación Lineal
* ejecutar_todas_pruebas.py - Script maestro (Windows/Linux)
* ejecutar_todas_pruebas.sh - Script maestro (Linux)
* pruebas/                  - Carpeta con casos de prueba
  - pruebas_memoria.txt
  - ejecutar_pruebas_memoria.py
  - pruebas_adicionales.py
  - comparar_algoritmos.py

FORMATO DE ENTRADA:
Bloques con:
  - Primera línea: número de participantes N
  - Siguientes N líneas: matriz N×N de conflictos (0-10)

SALIDA:
Tiempo de ejecución (ms), Número de nodos, Valor óptimo
